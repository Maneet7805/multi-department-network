# Multi-Department Network Design

A simulated multi-department network designed and configured in Cisco Packet Tracer.  
Includes DNS, DHCP, Web, Email, FTP, and IoT servers, with subnetting, VLANs, and routing applied.

## Technologies
- Cisco Packet Tracer
- Networking concepts: VLAN, Subnetting, Routing

## Features
- Multi-department network simulation
- DNS, DHCP, Web, Email, FTP, and IoT servers configured
- VLAN and routing implemented for optimized connectivity
- Test connectivity and server communication

## How to Run
1. Install Cisco Packet Tracer (https://www.netacad.com/courses/packet-tracer)
2. Open the `.pkt` file in Packet Tracer.
3. Run the simulation to test network functionality.

## Notes
- Designed for educational purposes and hands-on lab practice.
- VLANs and subnets are implemented for each department.
